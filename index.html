How to Use

Start by entering an HLS stream link into the input field labeled "Enter HLS Link". This can be a direct link to an .m3u8 playlist or other supported formats.

Select Player

Choose a player from the dropdown list (Cleper Player, JW Player, Video.js Player or any player). This selection determines how the HLS stream will be embedded.

Generate Preview

Click the "Generate Preview" button to create an iframe preview below. This iframe embeds the HLS stream using the selected player.

Copy Embed Link

Once the preview is generated, click the "Copy Embed Link" button to copy the embed code (iframe URL) to your clipboard. This link can then be used to embed the HLS stream into any HTML page or platform that supports iframes.

Embedding Methods

If your website is secured with an SSL certificate, use the HTTPS protocol for the embeddable URL. Ensure your streaming URL also supports the HTTPS format:

Example: https://noobromon.github.io/M3u8-To-iframe-And-Embet/live/player1.html?url=https://example.com/playlist.m3u8

Example of Embed Iframe:

<iframe 
        frameborder="0" 
        height="100%" 
        width="450px" 
        allowfullscreen="" 
        scrolling="no" 
        allow="autoplay;fullscreen" 
        src="https://noobromon.github.io/M3u8-To-iframe-And-Embet/live/player1.html?url=http://example.com/live.m3u8">
    </iframe>

HTTP Method (For Non-Secured Websites)

If your website does not have an SSL certificate, use the HTTPS protocol for embedding:

Example: https://noobromon.github.io/M3u8-To-iframe-And-Embet/live/player1.html?url=https://example.com/playlist.m3u8
